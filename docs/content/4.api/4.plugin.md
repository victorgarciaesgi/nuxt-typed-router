# Plugin

::note
To activate the plugin, you need to add `plugin: true` to your config options.
::

A plugin will be automatically generated, providing global access to `$typedRouter`, `$typedRoute` and `$routesNames`:

```ts
const { $typedRouter, $typedRoute, $routesNames } = useNuxtApp();
```
