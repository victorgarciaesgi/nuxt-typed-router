# Usage without autoImport

If you have configured your Nuxt app like this:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  imports: {
    autoImport: false
  }
})
```

It's still possible to use the typed `useRouter` and `useRoute`, but TypeScript limits the module override capability for `#app`.

Nuxt Typed Router will create an alias `@typed-router` where you can import router composables:

```ts
import {useRouter, useRoute, navigateTo} from '@typed-router';

const router = useRouter();

router.push({name: 'foo', params: { foo: 'bar' }});

navigateTo({name: 'foo', params: { foo: 'bar' }});

const route = useRoute('profile-id');
```
