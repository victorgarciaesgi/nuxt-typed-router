# ignoreRoutes

Allow to ignore selected files from being typed, for example 404 routes or catch-all routes.
You can pass an array of file paths (taking base of your `pagesDir`).

Usage:

```ts
ignoreRoutes: ["[...404].vue", "admin/[...slug].vue"];
```

## Type

`type: string[]`
`default: []`
