# plugin

`plugin` option let you enable [global utils](../2.usage/7.plugin.md)

## Type
 `type: boolean`  
 `default: false`

It allows you to use typed router in template as a global util.

 ```vue
<template>
  <div>{{ $routesNames }}</div>
</template>
```

```ts
const { $typedRouter, $typedRoute, $routesNames } = useNuxtApp();
```
