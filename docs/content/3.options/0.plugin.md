# plugin

`plugin` option lets you enable [global utils](/api/plugin).

## Type

`type: boolean`
`default: false`

It allows you to use the typed router in templates as a global util.

```vue
<template>
  <div>{{ $routesNames }}</div>
</template>
```

```ts
const { $typedRouter, $typedRoute, $routesNames } = useNuxtApp();
```
